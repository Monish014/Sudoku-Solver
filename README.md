# 🧩 Sudoku Solver (Python)

A simple and efficient Sudoku Solver built using Python. This application uses a backtracking algorithm to solve any valid 9×9 Sudoku puzzle accurately.

## 🚀 Features
- Solves 9x9 Sudoku puzzles
- Uses backtracking algorithm
- Accepts user input from console
- Displays solved Sudoku clearly
- Beginner-friendly implementation

## 🛠️ Tech Stack
- Python
- Recursion & Backtracking

## 📥 How to Run
1. Clone the repository
2. Open the project folder
3. Run the file:
   python sudoku.py

## 📌 Input Format
- Enter 81 numbers (row by row)
- Use 0 for empty cells

## 🧠 Algorithm
The solver uses backtracking:
- Finds empty cells
- Tries numbers from 1 to 9
- Checks row, column, and 3x3 grid validity
- Recursively solves the puzzle

## 📊 Time Complexity
Worst Case: O(9^(n*n))
